AgriguardAI is a **multi-functional smart agriculture system** designed to support farmers using **Artificial Intelligence, Machine Learning, and Deep Learning** techniques.  

The system integrates multiple intelligent modules to:

- 🌿 Detect plant diseases from images  
- 🌾 Recommend suitable crops  
- 📈 Predict crop yield  

This platform helps farmers make **data-driven decisions** to improve productivity and reduce crop loss.

---

# 🚀 Project Overview

AgriguardAI combines **computer vision** and **predictive analytics** to enhance modern farming practices.  

The platform enables:

- Early disease detection  
- Intelligent crop selection  
- Accurate yield estimation  
- Real-time prediction using trained AI models  

The application is built using **Flask**, providing an interactive web interface for users.

---

# 🎯 Key Features

## 🌿 Plant Disease Detection

- Detects plant diseases using leaf images  
- Uses **Convolutional Neural Networks (CNN)**  
- Provides real-time disease prediction  
- Helps reduce crop loss through early diagnosis  

---

## 🌾 Crop Recommendation

Suggests the best crops based on:

- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- Rainfall
- Soil pH

Uses **Machine Learning classification models** to generate recommendations.

---

## 📈 Crop Yield Prediction

- Predicts expected crop yield  
- Uses historical agricultural datasets  
- Applies **regression-based machine learning models**  
- Helps in planning agricultural production
